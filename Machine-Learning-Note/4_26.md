## 统计学习方法分类
**生成（概率）模型**，**判别（概率）模型**以及第三种与概率无关的**判别模型**  
**判别（概率）模型**与**生成（概率）模型**相比类似于黑匣子，准确率更强，解释性差一些
## SVM
**初始的三个简化条件**：二类，线性，可分
**margin**:点的margin,分类器的margin(所有训练集点的最小值)
注意**scale**
**Support Vector**
**优化问题**:一方面二次型是convex function,另一方面限定条件是线性的边界，是convex set
**拉格朗日乘子对于不等号的限定**：拉格朗日乘子大于等于0
**SVM对于不平衡问题有内在的机制**
